Example

This section of the tutorial helps you understand AHA! by creating your own example. We provide a step by step procedure that results in a fully functional example in the end.

The example will be a very simple HTML tutorial. (It was prepared by Hitesh Gupta as part of an internship.)

The creation of an AHA! application is a 4 step process

  • Create Conceptual Structure
  • Generate the Event Condition Access Rules or the Adaptation Rules
  • Writing Pages for the application
  • Define the look and feel of the application

It is advisable to see the installation instructions first, if you have not installed the AHA! system before.
