Using Claroline in GRAPPLE

GRAPPLE (and specifically the GRAPPLE Conversion Component module) can be installed on Claroline version 1.10. The integration requires two additional modules for Claroline:

  1. The first one is named "GRPPLAD" and allows the Claroline kernel to send data over .
  2. The second one named "GRAPPLE" is a fork from the Claroline Learning Path tool with additional features needed to send data to GRAPPLE.

Next the teacher can add a GRAPPLE course by creating a learning path in a Claroline course space. The Learning Path tool in Claroline allows the teacher to link resources (Claroline documents and exercises, GRAPPLE course, etc.) and define the path to navigate between them by branching conditions or prerequesites. To add a GRAPPLE course as a resource in a learning path, the teacher has to click on "Import a GRAPPLE course" to get a list of available GRAPPLE courses (the result of a "getCourses" request), choose the course he wants to add in this list and click on the import button:



Each resource in the course can be made available depending on the completion of other activities. Such sequencing is adaptation independent of the adaptation performed by .
