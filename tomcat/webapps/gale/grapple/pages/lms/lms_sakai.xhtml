Using Sakai in GRAPPLE

GRAPPLE (and specifically the GRAPPLE Conversion Component module) can be installed on Sakai versions 2.5 and 2.6. A tool imports all libraries automatically. You then need to edit the file "tomcat/sakai/sakai.properties" to indicate:

  • the location (URL) of ;
  • the URL where Sakai can register its listener on GEB:
    grapple.geb.registerListenerURL = http://localhost:9000/GrappleEventBus/gebListenerService?wsdl
  • the URL where Sakai can send events to:
    grapple.geb.sendEventURL = http://localhost:9000/GrappleEventBus/eventGEBListenerService?wsdl
To deploy this tool to Sakai you only have to execute "mvn clean install sakai:deploy" in the grapple directory. It will then copy the generated warfile to the tomcat/webapps directory and (re-)load itself.
