The Meta Info Tab of the Editor

The Meta Info Tab of the PRT Editor is shown below:

This tab consists of the following parts:

Describing the pedagogical meaning
At the top of the "Meta Info" tab the pedagogical meaning should be described. This is important information for course authors who might chose this PRT in the . On the other hand, this information is free text and will not be interpreted by any GRAPPLE component.
Additional Information
There is information on the creation time, modification time and author. This information cannot be changed in the tool. The information of the author is given as UUID, not as user name.
Defining a parameter in the PRT
The section on the parameter allows for defining a parameter for this PRT. The parameter has a value for which you define a default. The default value can be overwritten by a course author in the CAM for each individual instance of the PRT. The parameter can be used (using the "name") in the code section. The example figure shows a definition of a 'level' parameter with value 80. For the G-Prerequisite PRT this value is the knowledge level which is necessary so that concepts in the target socket become suitable. If an author uses this PRT in the CAM, she can change this level for each prerequisite.
