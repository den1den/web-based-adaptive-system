GALE Visual Authoring Tool

The Authoring Tool applet will appear in a separate window.
Note that the Authoring Tool requires the Sun Java plugin, version 1.5 or newer.
Please be patient, it may take a while to download all the classes of this applet.

Note also that the Authoring Tool is a signed applet. This is necessary in order to allow the tool to read and write your local file system and to transfer files between your local copy of your applications and the copy on the server.
