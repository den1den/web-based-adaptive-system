The AHA! Domain Model

Each AHA! application consists of a set of concepts, with associated attributes and rules. Most concepts are associated with pages.

In order to create the conceptual structure of the domain model we recommend that you use the graph author tool. Advanced authors may wish to have a look at the low level concept editor which allows complete control over the adaptive behavior of AHA!, but which is much more difficult to use.
