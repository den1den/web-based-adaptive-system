The CountModule (<count> tag)

The <count> is a special-purpose tag used mainly in a header to show how many pages (concept with a property "type" with value "page") the learner has read or still has to read in the current course. The two ways to use the tag are: 

  <gale:count 
    uri="gale://gale.tue.nl/cam/GRAPPLE"
    method="done"/>
and 
<gale:count 
  uri="gale://gale.tue.nl/cam/GRAPPLE"
  method="todo"/>
GALE does not really work with the notion of a "course" but counts the number of page concepts of which the URI starts with the given URI.
