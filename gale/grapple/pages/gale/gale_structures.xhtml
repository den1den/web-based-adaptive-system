Data Structures

GALE uses a strongly concept-oriented view on its applications (courses). All the data relating to a course are stored in what GALE calls the Domain Model. Adaptation in GALE is based on User Model updates. The schema of the UM (which concepts have which attributes) is part of the DM. The Adaptation Model is based on adaptation rules that are executed when events happen (on concepts or attributes). The adaptation rules are also part of the DM.

At runtime there is a per-user instantiation of the UM. This is the focus of our UM description.
