Using Moodle in GRAPPLE

GRAPPLE (and specifically the GRAPPLE Conversion Component module) can be installed on Moodle version 1.9. (The development was done with version 1.9.5. At the time of writing this tutorial the current version is 1.9.7.) The software can be found in the LMSconnectors directory of the GRAPPLE software repository. GRAPPLE Deliverable 5.2b contains detailed installation instructions that go beyond the scope of this tutorial. GRAPPLE Deliverable 7.5 also describes part of the installation.

The installation/configuration mainly consists of the following steps:

  • Configuring the Shibboleth based authentication service (which is already integrated in Moodle by default) to use the IdP (identity provider) of the federation of collaborating institutes and/or companies.
  • Copying code from the GRAPPLE repository to the "moodle/blocks" directory, logging in and clicking the "Notification" link to setup all the database tables.
